# Discord Music Bot

Private Discord music bot with slash commands support.

## Features
- Play music from YouTube
- Queue management
- Volume control
- DJ role permissions
- Both slash (/) and prefix (!) commands

## Deployment
Deployed on Koyeb with automatic Docker builds.
